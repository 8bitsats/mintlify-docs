## Installation

Install the FunPump SDK using npm or yarn.

<CodeGroup>

```bash npm
npm install @funpump/sdk
```

```bash yarn
yarn add @funpump/sdk
```

</CodeGroup>

## Initialize the SDK

Set up the SDK with your API key and Solana connection.

```typescript
import { Connection, clusterApiUrl } from '@solana/web3.js';
import { FunPumpSDK } from '@funpump/sdk';

const connection = new Connection(clusterApiUrl('mainnet-beta'));

const sdk = new FunPumpSDK({
  apiKey: 'your-api-key',
  connection,
  crossmintApiKey: 'your-crossmint-key', // optional
  phoenixEnabled: true, // optional
});
```

## Launch your first token

Create a token with a bonding curve in just a few lines of code.

```typescript
import { BN } from '@coral-xyz/anchor';
import { LaunchPlatform, CurveType } from '@funpump/sdk';

const { instruction, tokenMint } = await sdk.core.launchToken({
  creator: wallet.publicKey,
  name: 'My Token',
  symbol: 'MTK',
  uri: 'https://arweave.net/metadata',
  decimals: 9,
  supply: new BN(1_000_000_000).mul(new BN(10).pow(new BN(9))), // 1B tokens
  initialSolReserves: new BN(1).mul(new BN(10).pow(new BN(9))), // 1 SOL
  platform: LaunchPlatform.FunPump,
  curve: CurveType.ConstantProduct,
});

// Sign and send transaction
const tx = new Transaction().add(instruction);
await sendAndConfirmTransaction(connection, tx, [wallet, tokenMint]);
```

## Create an AI agent

Deploy an autonomous trading agent on Solana.

```typescript
const agent = await sdk.agent.createAgent(wallet.publicKey, {
  name: 'Alpha Trader',
  description: 'AI-powered arbitrage agent',
  autonomyLevel: 'high',
  capabilities: ['arbitrage', 'market_making'],
});

console.log('Agent created:', agent.agentId.toBase58());
```
