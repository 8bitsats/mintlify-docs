The FunPump SDK provides a comprehensive TypeScript API for building on Solana's most advanced token launch platform. This reference documents all available methods, parameters, and return types.

## SDK initialization

Initialize the SDK with your API key and Solana connection.

```typescript
import { Connection, clusterApiUrl } from '@solana/web3.js';
import { FunPumpSDK } from '@funpump/sdk';

const connection = new Connection(clusterApiUrl('mainnet-beta'));

const sdk = new FunPumpSDK({
  apiKey: 'your-api-key',
  connection,
  commitment: 'confirmed',
  crossmintApiKey: 'your-crossmint-key', // optional
  phoenixEnabled: true, // optional
});
```

## Configuration

<ParamField path="apiKey" type="string" required>
  Your FunPump API key.
</ParamField>

<ParamField path="connection" type="Connection" required>
  A Solana web3.js Connection instance.
</ParamField>

<ParamField path="commitment" type="Commitment">
  The commitment level for transactions (default: "confirmed").
</ParamField>

<ParamField path="crossmintApiKey" type="string">
  Your Crossmint API key for NFT minting.
</ParamField>

<ParamField path="phoenixEnabled" type="boolean">
  Enable Phoenix DEX integration (default: false).
</ParamField>

## SDK services

The SDK is organized into the following services:

### Core protocol

Access core protocol features for token launches, bonding curves, vaults, and vesting.

```typescript
sdk.core.launchToken(...)
sdk.core.buyTokens(...)
sdk.core.sellTokens(...)
sdk.core.getBondingCurve(...)
```

### Agent management

Create and manage autonomous trading agents.

```typescript
sdk.agent.createAgent(...)
sdk.agent.createAgentWithToken(...)
sdk.agent.deployStrategy(...)
```

### Phoenix DEX

Integrate Phoenix DEX market making capabilities.

```typescript
sdk.phoenixMM?.initializeMarketMaker(...)
sdk.phoenixMM?.deployAgentAsMarketMaker(...)
```

### Crossmint

Mint NFTs and tokens via Crossmint.

```typescript
sdk.crossmint?.createAgentNFT(...)
sdk.crossmint?.mintToken(...)
```

## Authentication

All API requests require a valid FunPump API key. Get your API key from the [FunPump dashboard](https://funpump.io/dashboard).

```typescript
apiKey: 'your-api-key'
```

## Error handling

The SDK throws typed errors for common failure cases.

```typescript
try {
  await sdk.core.launchToken(...);
} catch (error) {
  if (error instanceof InsufficientFundsError) {
    console.error('Insufficient funds:', error.message);
  } else if (error instanceof InvalidParameterError) {
    console.error('Invalid parameter:', error.message);
  } else {
    console.error('Unknown error:', error);
  }
}
```
