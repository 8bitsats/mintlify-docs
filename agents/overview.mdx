## Quick example

Create an autonomous trading agent in just a few lines of code.

```typescript
const agent = await sdk.agent.createAgent(wallet.publicKey, {
  name: 'Alpha Trader',
  description: 'AI-powered arbitrage agent',
  autonomyLevel: 'high',
  capabilities: ['arbitrage', 'market_making'],
});

console.log('Agent created:', agent.agentId.toBase58());
```

## Agent capabilities

Agents can be configured with various capabilities:

- **Arbitrage** - Exploit price differences across DEXes
- **Market making** - Provide liquidity and earn spreads
- **Yield farming** - Optimize yield across protocols
- **Auto compound** - Automatically compound rewards
- **Risk management** - Monitor and manage portfolio risk

## Autonomy levels

Choose the level of autonomy for your agent:

- **Low** - Manual approval required for all actions
- **Medium** - Automatic execution within predefined limits
- **High** - Full autonomy with risk management guardrails
