## Quick example

Create an agent NFT using Crossmint.

```typescript
const nft = await sdk.crossmint?.createAgentNFT(
  agent.agentId,
  'My Trading Agent',
  wallet.publicKey.toBase58()
);

console.log('NFT created:', nft.id);
console.log('NFT mint:', nft.onChain.mintAddress);
```

## Crossmint benefits

### Simplified minting

Crossmint handles all the complexity of NFT minting, including metadata upload, token creation, and distribution.

### Cross-chain support

Mint NFTs on multiple blockchains with a single API.

### Fiat payments

Accept fiat payments for NFT purchases via Crossmint's payment infrastructure.

### Custody solutions

Leverage Crossmint's custody solutions for secure asset management.
